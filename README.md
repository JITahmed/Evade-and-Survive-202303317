# Evade and Survive
A survival-based zombie shooter game built in Unity. Players navigate zombie-infested environments, fight off increasingly difficult waves of enemy types, and interact with a Trader NPC to exchange resources for weapons and supplies.
My Role — Character Designer
My responsibility on this project is the design of all in-game characters, covering every entity the player will encounter, fight, or interact with throughout the game. This includes all zombie enemy types, the playable survivor characters, and the Trader NPC.
What's in This Repository

Survivor character concept design sheets (2 playable characters)
